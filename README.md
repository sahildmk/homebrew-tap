# Homebrew Tap

Custom Homebrew tap for sahildmk's apps.

## Installation

```bash
brew tap sahildmk/tap
```

## Available Casks

### Quick Ports

A lightweight macOS menu bar app to monitor listening ports and kill processes.

```bash
brew install --cask quick-ports
```

Or install directly without tapping first:

```bash
brew install --cask sahildmk/tap/quick-ports
```
